# 🍽️ Smart Restaurant Ordering System

A full-stack restaurant ordering platform built with **Laravel 11** (backend) and **Flutter** (frontend).

## Features
- QR-based table sessions (persists across devices)
- Digital menu with categories, prep time, and availability
- Cart and unique order number system
- Real-time order tracking with Pusher
- Staff management panel
- AI-powered personalized recommendations

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Backend | Laravel 13 + MySQL |
| Frontend | Flutter (Web + Mobile) |
| Real-time | Pusher |
| Database | MySQL via Laragon |
